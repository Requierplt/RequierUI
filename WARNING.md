Из неё намеренно удалены ключевые компоненты, необходимые для полноценной работы приложения. В связи с этим возможны проблемы с отображением, некорректная работа элементов управления и отсутствие ожидаемой функциональности.

Репозиторий предназначен исключительно для ознакомления с визуальной составляющей проекта.
